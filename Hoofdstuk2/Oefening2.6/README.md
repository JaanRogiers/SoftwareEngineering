# Opgave 2.6
Heel vaak kun je de Label-control vervangen door een TextBlock-control.
Er is echter één groot verschil tussen deze twee controls.
Een Label heeft een Target-property waarmee je kunt aangeven welke control de focus krijgt wanneer een bepaalde toetsencombinatie gebruikt wordt.
Gebruik de Help-functie en zoek deze property op.
Bouw vervolgens een formulier met een TextBlock in het groen, een Label en een TextBox.
Zorg dat de cursor knippert in deze TextBox als de gebruiker de combinatie Alt-N indrukt.

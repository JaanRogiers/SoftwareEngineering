# Opgave 2.1
Creëer een nieuw project genaamd Demo en plaats drie knoppen op het formulier.
Zet de tekst ervan op respectievelijk 1, 2 en 3.
Maak Ook drie labels, met respectievelijk A, B en C erop.
Zet nu in de methoden behorend bij de knoppen een code die ertoe leid dat:
- het aanklikken van button1 de tekst van alle labels op Ja zet;
- het aanklikken van button2 de tekst van alle labels op Nee zet;
- het aanklikken van button3 de 'tesktwaarden' terugzet op A, B en C.

# Opgave 5.4
Schrijf een methode die een cirkel tekent bij gegeven coördinaten van het middelpunt en gegeven straal. De header moet zijn:
```csharp
private void DrawCircle
    (
        Canvas drawArea,
        SolidColorBrush brush,
        double xCentre,
        double yCentre,
        double radius
    )
{
    // Code here
}
```
Voorzie een drietal tekstvakken om het middelpunt en de straal in te geven. Na een druk op de knop verschijnt de cirkel.
